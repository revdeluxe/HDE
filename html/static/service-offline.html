LoRa Messaging App: Startup Routine

👤 Audience: Staff deploying LoRa hardware for field use

STEP 1: Launch the web backend

Open terminal and run:

sudo python main.py

You'll see "Listening on port 5000"

STEP 2: Verify LoRa Interface

The backend will automatically detect the LoRa interface.

Check the status bar for:

  • 👤 Username
  • 📶 RSSI
  • 📡 SNR
  • 🔄 RX Mode (should be ON)
  • 🗂️ TX Queue Depth
  • 🔄 Server State (should be "Listening")

If any status is missing, check the backend logs for errors.

STEP 3: Use the Web Interface

Open your browser and go to:

http://localhost:5000
  • Enter your name when prompted
  • Type a message and click Send
  • If everything works, the status circle turns ✅

STEP 4: Confirm Message Sync

Messages will be received on other units, and marked as "synced".

If no message is received:

  • Verify backend is running
  • Check that LoRa RX mode is ON (status bar)
